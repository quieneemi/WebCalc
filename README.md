# WebCalc
Web smart calculator. Graphs with chart.js
Calculations core written using c++ and imported as dynamic library

## Supported OS: Linux, Windows, MacOS

### Run from CLI:
> cd src/WebCalc.App

> dotnet restore && dotnet run

## Libraries & Frameworks
- ASP.NET Core MVC
- Chart.js
- Bootstrap
